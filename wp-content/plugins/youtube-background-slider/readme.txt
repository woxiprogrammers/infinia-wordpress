Showcase your YouTube videos on site background with a attractive video slider.

== Description ==

A Very effective and attractive and responsive YouTube Background slider plugin which makes your YouTube videos to work as slides in the background of your site to make it more attractive.  

* Can set Playlist option (want to show or not) for player  .  
* Can set Auto play for player to make it ON/OFF.  
* Can set random selection (first songs will be selected randomly on load) of songs.
* Can set Repeat playlist option (Yes/No).


Click Here for Demo http://wptreasure.com/downloads/youtube-background-slider

== Installation ==

* Upload the plugin folder to the /wp-content/plugins/ folder of your WordPress installation.  
* Activate the plugin.   
* You will see an option at the left side WordPress options option panel named "YouTube Background Slider".  
* Click on it to set the options.
Enjoy it......
